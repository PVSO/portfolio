Experiência

Disparidade Salarial no mercado de Dados

Out. 2025

Fui responsável pela análise da disparidade salarial entre Data Analysts e Data Scientists, utilizando o poder dos testes estatísticos para validar diferenças percebidas no mercado de dados. Para isso, utilizei um conjunto de dados exemplificativo do Kaggle sobre salários e empregos em Data Science, aplicando Análise Exploratória de Dados (EDA) e, de forma rigorosa, o Teste T de Welch (devido à heterogeneidade das variâncias). Os resultados indicaram que há uma diferença estatisticamente significativa nos salários médios, confirmando que a média salarial dos Data Scientists é superior à dos Data Analysts, reforçando a importância da metodologia estatística para conclusões precisas em estudos de mercado.

Stack: Python, Pandas, NumPy, Scipy, Seaborn, Plotly, Matplotlib.

7 dias de Pandas

Ago. 2025

Alura

Fui responsável pela análise de dados do uso dos acervos das bibliotecas da Universidade Federal do Estado do Rio Grande do Norte (UFRN), com o foco em identificar padrões de consumo de materiais por diferentes perfis de usuários (alunos, professores, pesquisadores e sociedade). Para a execução, utilizei dados públicos provenientes de registros de empréstimos físicos e implementei o projeto em Python com a estrutura CookieCutter Data Science, empregando Pandas para tratamento dos dados e Plotly para a criação de visualizações. Os resultados possibilitaram a identificação de oportunidades de melhoria no serviço e culminaram na disponibilização de um Dashboard interativo em Streamlit, essencial para apoiar a gestão do sistema de bibliotecas e fomentar o uso da informação como instrumento de geração de conhecimento.

Stack: Python, Pandas, Plotly, Streamlit, CookieCutter-Data-Science.
